@startuml

actor SalesPerson

SalesPerson -> System: Select food items and/or occasions
alt Food/occasions selected
    System -> System: Filter wines irrelevant to the selection
    
    alt No wine in output
        System -> SalesPerson: Notify no wine available
        SalesPerson -> System: Retry with different parameters
    else Wines available
        System -> SalesPerson: Provide wine recommendations
    end
else No selection
    System -> SalesPerson: Prompt for selection
end

@enduml
